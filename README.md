# Blackout Audio [ ⚫ ▶︎ ]

<img width="600" height="550" alt="Screenshot 2025-07-10 at 9 18 51 PM" src="https://github.com/user-attachments/assets/3eaf8306-7a49-4d4c-896b-202adaab0016" />

A web application that takes any uploaded audio file and generates a shareable web 

player in which the audio plays only when the listener's eyes are closed.

## Features

- Audio playback controlled by eye state (closed = play, open = pause)
- Calibration to customize eye closure detection to your eye shape
- File upload system for custom audio files
- Persistent audio storage using AWS S3
